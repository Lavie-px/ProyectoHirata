# ProyectoHirata
Dentro de la carpeta de docuementacion se encuentran el SQL de la base de datos sin datos
ademas de un pdf con la estructura de las tablas.
Para ocupar el programa es necesario contar con un programa que te permita levantar bases de datos
locales como Xampp o Wampp (Xampp no la recomiendo ya que se apropia de ciertos puertos) o contar con algun servicio
en la nube como BlueHosting, AWS, Azure o Google Cloud, dentro de la carpeta Modelo esta el archivo de conexion, deberas cambiar las variables UREL, USER y Clave
por las de tu base de datos
